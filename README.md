# SSUstagram

* 웹프로그래밍설계및실습 과목의 기말 프로젝트 입니다.
* Node.js 웹 애플리케이션 프레임워크인 Express를 사용했으며 Nunjucks 템플릿 엔진과 MySQL 데이터베이스 관리 시스템을 사용했습니다.
